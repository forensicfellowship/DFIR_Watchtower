# 🔭 DFIR Watchtower

Automated weekly intelligence digest for the DFIR community.
Live at: **[forensicfellowship.github.io/DFIR_Watchtower](https://forensicfellowship.github.io/DFIR_Watchtower/)**

---

## What It Is

DFIR Watchtower monitors the top DFIR blogs, threat intelligence feeds, and forensic tool update sources, then compiles everything into a single dark-mode HTML digest — published every Monday morning.


## Sections Covered

| Section | Color |
|---|---|
| Forensic Analysis | Teal |
| Threat Intelligence / Hunting | Red |
| Upcoming Events | Purple |
| Presentations / Podcasts | Blue |
| Malware | Amber |
| Miscellaneous | Green |
| Software Updates | Dark Blue |

## Structure

```
index.html                          ← always the latest digest (GitHub Pages entry point)
watchtower_digest_YYYY-MM-DD.html  ← dated archive copies
build_full_digest.py               ← digest builder (Python 3, stdlib only)
watchtower.py                      ← full multi-source scraper (local/cloud execution)
README.md
```

## How It's Built

The digest is generated by a scheduled Claude agent that:
1. Navigates to each source via browser
2. Extracts structured content (section headings + curated links)
3. Renders a self-contained dark-mode HTML file
4. Commits `index.html` to this repo → GitHub Pages publishes it automatically
